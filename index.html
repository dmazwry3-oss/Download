FAQ

Format URL apa saja yang didukung?

dmaz mendukung URL Scribd dengan path /document/, /doc/, atau /presentation/. Contoh: https://www.scribd.com/document/123456/Judul.

Apakah dmaz menyimpan URL saya?

Tidak. dmaz adalah situs statis tanpa server — semua proses dilakukan langsung di browser Anda. Tidak ada data yang dikirim ke server kami.

Bagaimana dmaz bekerja?

dmaz mengubah URL Scribd Anda menjadi URL pada layanan mirror (scribd.vdownloaders.com) yang menyediakan halaman download. dmaz hanya menjembatani — semua proses ekstraksi dilakukan oleh layanan mirror tersebut.

Saya mendapat error "URL tidak valid".

Pastikan URL diawali dengan https:// dan mengandung salah satu path yang didukung. Hindari URL hasil pencarian, profil, atau koleksi.
